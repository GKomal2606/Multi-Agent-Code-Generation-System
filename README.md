# Multi-Agent Code Generation System

An intelligent AI-powered code generation platform designed using Large Language Models (LLMs), multi-agent workflows, and Retrieval-Augmented Generation (RAG) techniques to generate context-aware and efficient code solutions.

## Overview

This project explores how multiple AI agents can collaboratively solve coding tasks by combining reasoning, retrieval, and generation capabilities. The system is designed to understand user prompts, retrieve relevant contextual information, and generate meaningful code outputs with improved accuracy and logical consistency.

The architecture integrates modern LLM orchestration techniques such as ReAct-based reasoning and vector-based semantic retrieval to create a more reliable and scalable AI-assisted development workflow.

## Key Features

- Multi-agent AI architecture for modular task handling
- ReAct-based reasoning workflows
- Retrieval-Augmented Generation (RAG) pipeline
- Semantic search using vector embeddings
- Context-aware code generation
- Prompt orchestration and intelligent response handling
- Scalable and modular Python-based design

## Technologies Used

- Python
- Large Language Models (LLMs)
- LlamaIndex
- ReAct Framework
- Mistral
- CodeLlama
- Vector Retrieval Pipelines

## System Workflow

1. User submits a coding query or prompt  
2. Relevant contextual information is retrieved using embeddings  
3. AI agents collaboratively process and reason through the request  
4. The LLM generates optimized code responses  
5. Final outputs are validated and returned to the user  

## Applications
